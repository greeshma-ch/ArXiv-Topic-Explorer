# ArXiv Topic Explorer

An NLP-powered research intelligence platform that analyzes ArXiv papers using BERTopic to discover emerging research trends and opportunities.

## Features

- Fetches papers directly from ArXiv
- Topic discovery using BERTopic
- Semantic embeddings using Sentence Transformers
- Topic trend analysis
- Research opportunity scoring
- Interactive Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- BERTopic
- Sentence Transformers
- Pandas
- Plotly

## Dataset

ArXiv Research Papers (via ArXiv API)

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
